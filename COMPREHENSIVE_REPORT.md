# 猫羽おかゆ 生日庆祝静态网页

---

## 第一部分：项目 README

### 1. 项目简介

本项目是为 **猫羽おかゆ** 制作的生日庆祝静态网页，旨在提供一个美观、互动性强的粉丝祝福平台。

**项目特点**：
-  纯 HTML + CSS + JavaScript 静态页面，无需构建工具
-  响应式设计，完美适配移动端和桌面端
-  精美的视觉设计，采用马卡龙色系
-  丰富的交互功能，包括弹幕、留言板、时间线等

### 3. 核心功能

| 页面 | 功能 | 说明 |
|------|------|------|
| Page 1 | 首页 | 生日祝福主页面，包含倒计时、头像展示 |
| Page 2 | 视频区 | 精选视频切片展示，支持弹窗播放 |
| Page 3 | 照片墙 | 照片轮播展示，支持自动播放 |
| Page 4 | 时间线 | 重要事件时间线，点击可查看详情 |
| Page 5 | 留言板 | 粉丝留言展示，支持分页浏览 |
| Page 6 | 小说区 | 中日双语对照小说，hover联动高亮 |
| Page 7 | 致谢页 | 制作人员信息和版权声明 |

### 4. 技术实现

#### 4.1 主要组件

| 组件 | 功能描述 |
|------|---------|
| **MessageBoardRenderer** | 留言板渲染器，支持分页、瀑布流布局、XSS防护 |
| **DanmakuManager** | 弹幕管理器，固定6条轨道均匀分布 |
| **DanmakuAutoPlayer** | 弹幕自动播放器，随机发送留言内容 |
| **NovelSyncController** | 小说同步控制器，实现中日段落联动高亮 |
| **TimelineImageViewer** | 时间线图片查看器 |
| **MusicPlayer** | 背景音乐播放器 |

#### 4.2 关键技术

- **中日双语对齐**：采用构建期预对齐方案，生成 `novel_data.json`，运行时通过 `data-id` 直查
- **XSS防护**：使用 `textContent` 和 `Utils.escapeHtml()` 处理用户输入
- **响应式设计**：使用媒体查询适配不同屏幕尺寸
- **事件委托**：统一事件监听，支持动态内容
