# 猫羽おかゆ お誕生日 🎂

一个为 **猫羽おかゆ** 制作的生日庆祝静态网页。

## 项目结构

```
sssdsd/
├── index.html              # 主页面
├── README.md               # 项目说明
├── .gitignore              # Git 忽略规则
├── assets/
│   ├── images/             # 图片资源
│   └── videos/             # 视频资源
├── novel_jp.txt            # 日文小说
└── novel_zh.txt            # 中文小说翻译
```

## 说明

- 纯 HTML + CSS 静态页面，无需构建工具
- 使用 Google Fonts（Noto Sans JP）和 LXGW WenKai 字体
- 响应式设计，适配移动端
